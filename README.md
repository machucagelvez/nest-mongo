# Ejecutar en desarrollo

1. Clonar repositorio

2. Ejecutar

```
yarn install
```

3. Tener Nest CLI instalado

```
yarn install
```

4. Levantar la BD

```
docker-compose up -d
```

5. Reconstruir la BD

```
localhost:3000/api/v1/seed
```

## Stack

- Nest
- MongoDB
